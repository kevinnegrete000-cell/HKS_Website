HKS DIGITAL ARCHIVE — 3D OBJECT TEST

1. Open this entire folder in Visual Studio Code.
2. Run index.html with the Live Server extension. Do not open it directly as file://.
3. Click the new “Window Profile Prototype” node in the Geometry Experiments orbit.
4. The observatory will zoom to the node, then open the textured OBJ viewer.
5. Drag to orbit, scroll to zoom, and close the panel to return to the observatory.

Repository structure to preserve:
- index.html
- style.css
- script.js
- assets/window-profile/window-profile.obj
- assets/window-profile/window-profile.mtl
- assets/window-profile/textures/diffuse.jpg
- assets/window-profile/textures/normal.jpg
- assets/window-profile/textures/ao.jpg

Three.js is loaded from jsDelivr through the import map in index.html.
