    "RUN LOCALLY
"
    "1. Open this entire folder in Visual Studio Code.
"
    "2. Right-click index.html.
"
    "3. Choose Open with Live Server.

"
    "Do not open index.html directly with file://. ES modules and OBJ/texture loading require a local web server.

"
    "REPOSITORY
"
    "Upload index.html, style.css, script.js, models-manifest.json, and the complete assets folder.
"
    "Do not rename or move files inside assets/models unless you also update the corresponding paths in script.js.

"
    "MODEL INTERACTION
"
    "Click a node → constellation focuses on it → information panel opens → textured OBJ auto-rotates.
"
    "Drag the model to orbit manually and scroll to zoom. Close the panel to return to the constellation.
